# ContentSale
内容销售系统
采用前后端分离的模式
前端采用react+antDesign
后端采用springBoot
## front-end文件夹内的是前端代码
### 前端初始化项目过程
1. npm升级到最新版本 
```
$ npm install -g npm
```
2. 在国内，你可以安装 cnpm 获得更快速、更安全的包管理体验。使用如下命令安装：
```
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
```
3. 安装前端代码依赖的包
```
$ cnpm install
```
4. 编译并运行前端项目
```
$ cnpm run dev
```
